## Relational Schema

The complete relational schema covers 11 tables: `user_account`, `user_profile`, `skill_category`, `skill_offering`, `availability_slot`, `session_request`, `session`, `message_thread`, `message`, `review`, `report`. Full schema is in `db/schema.sql`.
